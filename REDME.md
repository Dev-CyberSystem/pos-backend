# POS Backend (MERN)

Backend para un sistema de ventas (POS) con:
- Productos + control de stock
- Ventas + emisión de ticket
- Caja por turnos (mañana / tarde) con apertura y cierre
- Reportes diarios y mensuales

## Requisitos
- Node.js 18+ (recomendado 20+)
- MongoDB (local o cloud)

## Instalación
```bash
npm install
